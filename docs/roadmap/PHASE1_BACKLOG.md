## 1. 概述

本文档定义 Phase 1（v1.0）的 **端到端切片（Slices）**，是 FRD 生成和开发任务的输入。

### 切片原则

1. **一个切片 = 一条可验收的端到端路径**
2. 切片按开发顺序排列，后续切片可能依赖前置切片
3. 每个切片有明确的 Feature ID 映射和验收映射
