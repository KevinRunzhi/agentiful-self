# AgentifUI Phase 1 验收标准

* **版本**：v0.1
* **最后更新**：2026-01-27
* **关联文档**：[ROADMAP_V1_0.md](./ROADMAP_V1_0.md)

## 1. 概述

本文档定义 Phase 1（v1.0）各阶段的验收标准（Acceptance Criteria），是切片完成度的唯一判定依据。

### 验收原则

1. **端到端可验证**：每个验收条目必须对应可执行的测试用例
2. **渐进式累积**：后阶段必须保持前阶段能力不退化
3. **可度量**：包含明确的性能指标和超时阈值
