# AgentifUI 降级矩阵 v0

* **规范版本**：v0.1
* **最后更新**：2026-01-26
* **状态**：草稿，待评审
* **参考**：PRD 第 5.5 节

---

## 1. 概述

本文档定义 AgentifUI 在后端编排平台不可用或能力不支持时的降级策略，确保关键功能持续可用。

### 设计原则

1. **优雅降级**：核心功能不可用时，提供明确的用户反馈
2. **部分可用**：保持非依赖后端的功能正常运行
3. **可追溯**：降级事件记录审计日志
4. **可恢复**：后端恢复后自动恢复功能

---

## 2. 降级场景矩阵

### 2.1 后端编排平台不可用

**触发条件**：网关无法连接后端编排平台（Dify/Coze/n8n）

| 功能类别 | 功能 | 状态 | 用户体验 |
|----------|------|------|----------|
| **认证** | 登录/登出 | ✅ 可用 | 正常 |
| **认证** | Token 刷新 | ✅ 可用 | 正常 |
| **导航** | 页面跳转 | ✅ 可用 | 正常 |
| **导航** | 应用列表浏览 | ✅ 可用 | 正常 |
| **历史** | 查看对话历史 | ✅ 可用 | 正常 |
| **历史** | 查看执行记录 | ✅ 可用 | 正常 |
| **统计** | 查看使用统计 | ✅ 可用 | 正常 |
| **审计** | 查看审计日志 | ✅ 可用 | 正常 |
| **设置** | 用户设置 | ✅ 可用 | 正常 |
| **设置** | 管理后台 | ✅ 可用 | 正常 |
| **生成** | 发送消息 | ❌ 不可用 | 显示不可用提示 |
| **生成** | 运行工作流 | ❌ 不可用 | 显示不可用提示 |
| **生成** | 停止生成 | ❌ 不可用 | 按钮禁用 |

**用户提示**：

```
AI 服务暂时不可用，请稍后重试。您仍可查看历史记录和管理设置。
```

---

### 2.2 后端不支持停止

**触发条件**：后端编排平台不支持任务取消 API

| 功能 | 降级行为 | 用户体验 |
|------|----------|----------|
| 停止生成按钮 | 可点击，执行软停止 | 正常显示 |
| 停止效果 | 立即停止前端渲染 | 流式输出停止 |
| 后端行为 | 继续执行直至完成 | 提示说明 |
| 计费 | 按实际执行计费 | 无法节省 |

**用户提示**：

```
已停止显示。后端可能仍在执行，Token 消耗将按实际计算。
```

**API 响应**：

```json
{
  "result": "success",
  "stop_type": "soft"
}
```

---

### 2.3 后端不支持引用来源

**触发条件**：后端响应中不包含 `citations` 字段

| 功能 | 降级行为 | 用户体验 |
|------|----------|----------|
| 引用来源区域 | 隐藏 | 不显示引用区域 |
| 引用链接 | 不可用 | 无法跳转来源 |

**UI 状态**：引用区域完全隐藏，不显示任何提示。

---

### 2.4 后端不支持工具调用

**触发条件**：后端不支持 `tools` / `function_call` 能力

| 功能 | 降级行为 | 用户体验 |
|------|----------|----------|
| 工具调用展示 | 降级为文本提示 | 显示说明 |
| 交互式工具 | 不可用 | 按钮禁用 |
| 结构化输出 | 展示为纯文本 | 格式较简 |

**用户提示**：

```
此应用的工具调用功能暂不可用，仅支持文本回复。
```

---

### 2.5 后端不支持流式响应

**触发条件**：后端仅支持阻塞式响应

| 功能 | 降级行为 | 用户体验 |
|------|----------|----------|
| 流式输出 | 等待完整响应后一次性显示 | 等待时间较长 |
| 加载状态 | 显示加载动画 | 持续转圈 |
| 停止按钮 | 根据后端能力决定 | 可能不可用 |

**用户提示**：

```
正在生成回复，请稍候...
```

---

### 2.6 后端不支持 Human-in-the-loop

**触发条件**：后端不返回 HITL 结构化事件

| 功能 | 降级行为 | 用户体验 |
|------|----------|----------|
| 确认对话框 | 不显示 | 自动继续或失败 |
| 选项选择 | 不显示 | 无法交互 |
| 审批流程 | 不可用 | 提示说明 |

**用户提示**：

```
此应用不支持交互式审批功能。
```

## 3. 降级检测机制

### 3.1 能力发现

应用注册时，网关探测后端能力并缓存：

```typescript
interface AppCapabilities {
  streaming: boolean;      // 支持流式
  stop: boolean;           // 支持停止
  tools: boolean;          // 支持工具调用
  files: boolean;          // 支持文件上传
  citations: boolean;      // 支持引用来源
  hitl: boolean;           // 支持 HITL
}
```

### 3.2 实时检测

网关通过以下方式检测后端状态：

| 检测方式 | 频率 | 用途 |
|----------|------|------|
| 健康检查 | 30s | 检测可用性 |
| 超时检测 | 每请求 | 检测响应能力 |
| 错误计数 | 滑动窗口 | 熔断判断 |

### 3.3 熔断策略

| 指标 | 阈值 | 动作 |
|------|------|------|
| 错误率 | > 50% (1min) | 触发熔断 |
| 响应超时 | > 30s | 请求失败 |
| 熔断恢复 | 30s 后半开 | 探测恢复 |

## 4. 前端降级 UI 规范

### 4.1 不可用状态

**输入区域**：

```tsx
<ChatInput disabled>
  <DisabledOverlay>
    <WarningIcon />
    <span>AI 服务暂时不可用</span>
    <RetryButton onClick={retry}>重试</RetryButton>
  </DisabledOverlay>
</ChatInput>
```

### 4.2 软停止提示

```tsx
<Toast type="warning">
  已停止显示。后端可能仍在执行，Token 消耗将按实际计算。
</Toast>
```

### 4.3 能力不可用提示

```tsx
<FeatureDisabledBanner>
  <InfoIcon />
  <span>此应用不支持引用来源功能</span>
</FeatureDisabledBanner>
```

## 5. 监控与告警

### 5.1 监控指标

| 指标 | 类型 | 告警阈值 |
|------|------|----------|
| `gateway.backend.available` | Gauge | = 0 |
| `gateway.backend.error_rate` | Gauge | > 10% |
| `gateway.degradation.active` | Gauge | > 0 |
| `gateway.soft_stop.count` | Counter | - |

### 5.2 审计事件

降级触发时记录审计事件：

```json
{
  "action": "gov.degradation.triggered",
  "resource_type": "app",
  "resource_id": "app_xxx",
  "metadata": {
    "degradation_type": "backend_unavailable",
    "backend_platform": "dify",
    "duration_seconds": 120
  }
}
```
