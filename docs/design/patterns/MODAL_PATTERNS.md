# 模态框交互规范

> **版本**：v0.1  
> **最后更新**：2026-01-27  
> **引用**：DESIGN_SYSTEM_P1 v0.1

---

## 概述

本文档定义 AgentifUI 中模态框（Modal/Dialog）的交互规范。

---

## 类型

| 类型 | 用途 | 尺寸 |
|------|------|------|
| Alert | 简单确认/提示 | sm (400px) |
| Confirm | 确认操作 | sm (400px) |
| Form | 表单编辑 | md (540px) |
| Detail | 详情展示 | lg (720px) |

---

## 结构

```
┌─ Header ─────────────────────────┐
│ Title                      [×]   │
├─ Body ───────────────────────────┤
│                                  │
│              内容区              │
│                                  │
├─ Footer ─────────────────────────┤
│             [取消] [确认]        │
└──────────────────────────────────┘
```

---

## 交互规则

### 打开

- 使用淡入 + 缩放动画
- 焦点移动到模态框内

### 关闭

- 点击关闭按钮：关闭
- 点击遮罩：关闭（可配置禁用）
- 按 ESC 键：关闭（可配置禁用）
- 表单类需确认放弃编辑

### 焦点管理

- 打开时焦点移到第一个可聚焦元素
- 使用 Tab 键在模态框内循环
- 关闭后焦点返回触发元素

---

## 按钮规范

### 布局

- 右对齐：`[取消] [主操作]`
- 主操作按钮在右侧

### 命名

| 操作类型 | 主按钮文案 |
|----------|------------|
| 创建 | 创建 / 添加 |
| 编辑 | 保存 / 更新 |
| 删除 | 删除（destructive 样式） |
| 确认 | 确认 / 确定 |

---

## 嵌套限制

- 最多允许两层嵌套
- 避免模态框中再打开全屏模态框
