# 表单交互规范

> **版本**：v0.1  
> **最后更新**：2026-01-27  
> **引用**：DESIGN_SYSTEM_P1 v0.1

---

## 概述

本文档定义 AgentifUI 中表单相关的交互规范和最佳实践。

---

## 表单布局

### 标签位置

| 布局 | 适用场景 |
|------|----------|
| Top-aligned（顶部对齐） | 默认布局，适合大多数表单 |
| Left-aligned（左对齐） | 空间充足时，提高扫描效率 |
| Inline（行内） | 简单表单，如搜索框 |

### 表单宽度

- 单列表单：最大宽度 480px
- 双列表单：最大宽度 720px
- 表单容器居中或左对齐（根据页面布局）

---

## 校验规则

### 校验时机

| 时机 | 说明 |
|------|------|
| onChange | 用户输入时实时校验（仅格式校验） |
| onBlur | 失焦时校验（完整校验） |
| onSubmit | 提交时全量校验 |

### 校验反馈

- 即时反馈：输入框边框变色 + 显示错误信息
- 延迟反馈：避免用户输入过程中频繁提示（debounce 300ms）
- 成功反馈：可选显示绿色勾号

---

## 提交行为

### 提交按钮状态

| 状态 | 条件 | UI 表现 |
|------|------|---------|
| Disabled | 必填字段未填写 | 按钮禁用 |
| Active | 所有必填字段已填写 | 按钮可点击 |
| Loading | 提交中 | 按钮显示 Spinner，禁用 |

### 提交成功

- 显示 Toast 成功提示
- 跳转到目标页面 或 关闭模态框
- 清空表单（如需继续录入）

### 提交失败

- 显示 Toast 错误提示
- 保留表单数据
- 定位到第一个错误字段

---

## 特殊字段

### 密码字段

- 提供显示/隐藏切换按钮
- 显示密码强度指示器（可选）
- 确认密码字段实时校验匹配

### 文件上传

- 支持拖拽上传
- 显示文件名和大小
- 提供删除/替换操作
- 显示上传进度
